🚀 Control por Gestos Espacial

Navega por el espacio usando gestos con tu cámara

🎮 Controles de Gestos:

  • 👆 Apunta con el dedo índice = El personaje te sigue como cursor
  • ✋ Mano abierta = Pausa/Reanuda el juego
  • 🎯 Mueve el dedo = El personaje se mueve hacia donde apuntas

🎹 Controles de Teclado (Alternativos):

  • WASD o Flechas = Movimiento tradicional
  • Espacio = Reiniciar cuando mueres
